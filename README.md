# 病院内でのローカル LLM 活用 プレゼンテーション

このリポジトリは、病院内でのローカル LLM 活用に関するプレゼンテーション資料を含んでいます。

## 必要条件

- Node.js (v14 以上)
- npm

## セットアップ

```bash
npm install
```

## 使用方法

### プレゼンテーションの表示

```bash
npm start
```

### PDF の生成

```bash
npm run build
```

## ファイル構成

- `presentation.md`: プレゼンテーションのメインコンテンツ
- `package.json`: プロジェクト設定と依存関係
